在此示例中，我们使用Vashishta势搜索小的SiO₂团簇。

````bash
$ cat SiO2.cell

%BLOCK LATTICE_CART
20 0 0
0 20 0
0 0 20
#FIX
%ENDBLOCK LATTICE_CART

#CLUSTER
#ELLIPSOID=2.5 0.75

#SPECIES=Si%NUM=1,O%NUM=2
#NFORM=4

#MINSEP=1.0 Si-Si=3.00 Si-O=1.60 O-O=2.58
````

在指定的最小距离约束下构建一个4 × SiO₂的随机团簇，并将其限制在椭球体内，椭球体体积为4/3 × π × r³，其中r = 2.5 Å。第二个数字（在本例中为0.75）描述椭球体平均的球形程度。值0.0强制球形。
