**`B.cell`:**

```
%BLOCK LATTICE_ABC
5.0544 5.6199 6.9873
90 90 90
#FIX
%ENDBLOCK LATTICE_ABC

#SPECIES=B
#NATOM=28
#MINSEP=1.7

KPOINTS_MP_SPACING 0.1

FIX_ALL_CELL : true

%BLOCK SPECIES_POT
B 2|1.6|7|7|9|20:21(qc=4)
%ENDBLOCK SPECIES_POT
```

这是γ-B的固定单位晶胞搜索，使用实验性晶胞参数和28个B原子。

$ despawn
$ tidy.pl

压力为负是由于k点和基组收敛不好造成的。但对于固定晶胞搜索，能量景观足以快速定位γ-B的已知`Pnnm`结构。在本例中，它在1053次尝试中被定位了4次。在较长的运行中，它在21,222次尝试中被定位了113次，这表示识别基态的平均尝试次数为188。
