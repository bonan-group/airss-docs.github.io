在本示例中，我们将探索在100 GPa（或1 MBar）压力下碳的能量景观。作为参考，地球中心的压力约为350 GPa。

````bash
$ cat C2.cell

%BLOCK LATTICE_CART
1.709975 0 0
0 1.709975 0
0 0 1.709975
%ENDBLOCK LATTICE_CART

#VARVOL=5

%BLOCK POSITIONS_FRAC
C 0.0 0.0 0.0 # C1 % NUM=1
C 0.0 0.0 0.0 # C2 % NUM=1
%ENDBLOCK POSITIONS_FRAC

#MINSEP=1.3
````

cell文件中唯一的AIRSS相关命令是`#MINSEP=1.3`。这对于轻元素来说不是必需的，但对于过渡金属来说，避免核心重叠以防止电子结构收敛不良很重要。

搜索与[示例2.1]({{ '/zh/tutorials/2.1/' | relative_url }})中执行的搜索相同，除了这次搜索将使用VASP代码进行。首先创建一个`C2.POTCAR`文件。airss.pl脚本的使用方式通常相同，除了设置`-vasp`标志。


**`C2.INCAR`:**

```
ENCUT  = 300
ALGO  = Conjugate
ISYM = 2
NELM = 100        #need to be increased in some cases
EDIFF  = 1E-05
PREC = Accurate
ISMEAR = 0
SIGMA=0.02
LORBIT = 11

# Geometry optimization
IBRION = 1
ISIF   = 3       # Relax ions only (2), relax everything (3)
NSW    = 500     # Number of ionic steps
EDIFFG = -1E-02  # Force convegence condition
NFREE  = 10
POTIM  = 0.2

# I/O control
LCHARG = .FALSE.  # Do not write out charge density
LWAVE  = .FALSE.  # Do not write wave function file

KSPACING = 0.7
```

````bash
$ airss.pl -vasp -press 100 -max 20 -seed C2
````
