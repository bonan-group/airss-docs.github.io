在此示例中，我们将使用`-gulp`和Vashishta势对二维二氧化硅结构进行搜索。首先查看buildcell的输入文件：

````bash
$ cat SiO2.cell

%BLOCK LATTICE_CART
1 0 0
0 1 0
0 0 15
#CFIX
%ENDBLOCK LATTICE_CART

#TARGVOL=50

#SLAB
#WIDTH=4.0
#SHIFT=7.5

#SPECIES=Si%NUM=1,O%NUM=2
#NFORM=4

#MINSEP=1.0 Si-Si=3.00 Si-O=1.60 O-O=2.58

#COMPACT
````

晶胞矢量被指定为固定c轴长度（15 Å）。a轴和b轴的长度（和方向）将由`buildcell`选择，使得整个单胞的体积为50 Å³（由`TARGVOL=50`设置）。对于给定数量的原子，`TARGVOL`的较大值将导致更薄的层。

SLAB设置指示`buildcell`使用适合这种几何形状的内部设置。随机结构由4个SiO₂公式单位构建，满足指定的最小分离距离。它们被垂直于c轴的硬墙平面所限制，间距为4 Å。生成的平板的中心被移动7.5 Å，使其位于单胞的中心。
