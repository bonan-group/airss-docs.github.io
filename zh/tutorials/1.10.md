**下载输入文件：**
[<i class="fas fa-download"></i> ABC.cell]({{ '/assets/examples/1.10/ABC.cell' | relative_url }}){: .btn .btn--primary .btn--small}
[<i class="fas fa-download"></i> ABC.pp]({{ '/assets/examples/1.10/ABC.pp' | relative_url }}){: .btn .btn--primary .btn--small}

现在我们执行三元搜索。


**`ABC.cell`：**

```
#VARVOL=15
#SPECIES=A,B,C
#NATOM=2-8
#MINSEP=1.5
```

**`ABC.pp`：**

```
3 12 6 5
A B C
# Epsilon
1.00 1.50 1.25
0.50 0.75
0.60
# Sigma
2.00 1.60 1.75
1.76 1.5
1.80
```

```console
$ airss.pl -pp3 -max 1000 -seed ABC
```

输出可以使用cryan进行分析。`ca -s`将给出所覆盖组分的摘要，`ca -m`计算凸包。两者都会生成大量输出，所以这里我们只考虑凸包上的那些组分，使用`-de 0`标志，将能量窗口设置为零。
