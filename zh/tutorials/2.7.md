在此示例中，我们搜索堆积良好的甲烷(CH₄)分子晶体。使用Tersoff势确定原子间相互作用，预计它只能提供粗略的近似。

````bash
$ cat CH4.cell

#TARGVOL=7.69

%BLOCK POSITIONS_ABS
    C   0.86267   0.87843   0.69644 # 1-Td % NUM=1
    H  -0.15687   1.20067   0.63115 # 1-Td
    H   1.19106   0.93720   1.71422 # 1-Td
    H   0.93994  -0.13355   0.35439 # 1-Td
    H   1.47618   1.50890   0.08615 # 1-Td
%ENDBLOCK POSITIONS_ABS

#SYMMOPS=4
#SGRANK=20
#SLACK=0.25
#OVERLAP=1
##SYSTEM=Cubi
#MINSEP=1.0 C-C=2.10 C-H=1.45 H-H=1.00
````

使用此输入文件，`buildcell`将尝试构建一个随机单胞，从分子晶体最常见的前20个空间群中随机选择四个对称操作符的空间群。CH₄四面体单位（或分子）以绝对坐标指定。所有原子以相同的方式标记(`1-Td`)以将原子绑定到一个单位中。最小距离约束首先通过移动分子来应用，使用`SLACK`参数0.25（`MINSEP`距离乘以0.75）。一旦满足这些放宽的约束条件，将应用使用指定`MINSEP`距离硬球势的优化，并允许单位的旋转。最后允许`OVERLAP`为1。这可以降低到0.1或更低，但计算成本更高。
