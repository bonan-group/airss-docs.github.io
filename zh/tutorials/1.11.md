**`AB.cell`：**

```
%BLOCK LATTICE_ABC
3.27 3.27 5.35
90 90 120
#FIX
%ENDBLOCK LATTICE_ABC

%BLOCK POSITIONS_FRAC
A 0.3333333333 0.6666666667 0.25 # A1 % NUM=1 POSAMP=0 NOMOVE
A 0.6666666667 0.3333333333 0.75 # A2 % NUM=1 POSAMP=0 NOMOVE
A 0.5000000000 0.5000000000 0.50 # A3 % NUM=5 POSAMP=4 ADATOM
B 0.5000000000 0.5000000000 0.50 # B1 % NUM=4 POSAMP=4 ADATOM
%ENDBLOCK POSITIONS_FRAC

#SUPERCELL=3 -3 0 5 5 0 0 0 3
#HOLE=4
#HOLEPOS=0.5 0.5 0.5
#SLACK=0.25
#MINSEP=2.5
#OVERLAP=0.1

FIX_ALL_CELL : true
```

**`AB.pp`：**

```
2 12 6 5
A B
# Epsilon
1.0 -1
0.5
# Sigma
3.0 -1
2.0
```

在这里，我们通过在块晶体中挖掘孔洞并随机化孔洞中原子的位置来搜索缺陷。对于这些更复杂的几何结构，在优化前可视化随机化阶段的输出特别重要。

```console

$ airss.pl -pp3 -max 100 -seed AB
$ ca -r -t 10
```
