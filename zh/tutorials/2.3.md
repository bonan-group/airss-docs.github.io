在此示例中，我们使用配位约束来快速定位二十面体C₂₀富勒烯。

````bash
$ cat C.cell

%BLOCK LATTICE_CART
20 0 0
0 20 0
0 0 20
#FIX
%ENDBLOCK LATTICE_CART

%BLOCK POSITIONS_FRAC
C 0.0 0.0 0.0 # C1 % NUM=20
%ENDBLOCK POSITIONS_FRAC

#SYMMOPS=1
#NFORM=1

#SPHERE=2.1
#POSAMP=2.5

#CLUSTER

#MINSEP=1.44
#COORD=3
#MINBANGLE=91

FIX_ALL_CELL : true
````

在上述配置中，20个碳原子最初随机放置在以原点为中心、半径为2.5 Å的球体内。
随后它们被限制在半径为2.1 Å的球体内，使得最小分离距离为1.44 Å，
所有原子连接到三个其他原子，最小键角为91°（以排除四元环）。
