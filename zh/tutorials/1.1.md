**下载输入文件：**
[<i class="fas fa-download"></i> Al.cell]({{ '/assets/examples/1.1/Al.cell' | relative_url }}){: .btn .btn--primary .btn--small}
[<i class="fas fa-download"></i> Al.pp]({{ '/assets/examples/1.1/Al.pp' | relative_url }}){: .btn .btn--primary .btn--small}

在这个例子中，我们将使用随机搜索来寻找Lennard-Jones固体的基态。

```console
$ ls

Al.cell	Al.pp	README

$ cat Al.cell

%BLOCK LATTICE_CART
2 0 0
0 2 0
0 0 2
%ENDBLOCK LATTICE_CART

%BLOCK POSITIONS_FRAC
Al 0.0 0.0 0.0 # Al1 % NUM=8
%ENDBLOCK POSITIONS_FRAC

#MINSEP=1.5
```

`Al.cell` 是"种子"输入文件，它描述了应如何构建随机结构（由 `buildcell` 进行）。在这种情况下，8个原子被随机放置在一个形状随机的晶胞中，体积约为每个原子8 Å³（变化范围在5%以内）。如果请求更多的原子，晶胞的体积将相应地缩放。任何原子间距小于1.5 Å的配置都将被拒绝。

**`Al.pp`：**

```
1 12 6 2.5
Al
# Epsilon
1
# Sigma
2
# Beta
1
```

`Al.pp` 文件包含对势的参数。现在我们进行搜索：

```console
$ airss.pl -pp3 -max 20 -seed Al
```

执行AIRSS perl脚本（`airss.pl`）。选择 `-pp3` 对势代码进行结构优化（默认值为CASTEP），并请求20个松弛计算（默认值是一个很大的数字——需要用户手动停止脚本）。现在可以分析结果。

上述是根据能量（严格来说是焓）对结果进行的"排名"。列的含义为：（1）唯一结构名称；（2）压力（GPa）；（3）体积（Å³ 每个化学式单位）；（4）第一个结构的焓（eV 每个化学式单位），然后是相对于该结构的值；（5）化学式单位的数量；（6）化学式；（7）对称性；（8）重复次数。

对于这样一个简单的系统，你很可能在这20次尝试中找到HCP基态（空间群 `P63/mmc`）。当然，这是一种随机方法，你也可能很不幸运。如果是这样，只需再次运行AIRSS脚本。
