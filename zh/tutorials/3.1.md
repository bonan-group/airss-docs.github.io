在本示例中，我们将探索在100 GPa（或1 MBar）压力下碳的能量景观。作为参考，地球中心的压力约为350 GPa。

本示例是[示例2.1]({{ '/zh/tutorials/2.1/' | relative_url }})的DFT类似物。我们生成碳结构，并使用CASTEP代码对其进行松弛。

> **注意:** 要运行本示例，您需要安装`castep`二进制文件并将其添加到您的路径中。此代码*不会*自动作为AIRSS包的一部分安装。有关获取CASTEP的信息，请参阅[http://www.castep.org/CASTEP/GettingCASTEP](http://www.castep.org/CASTEP/GettingCASTEP)。

## 输入文件

检查seedfile `C2.cell`:

```console
$ cat C2.cell

%BLOCK LATTICE_CART
1.709975 0 0
0 1.709975 0
0 0 1.709975
%ENDBLOCK LATTICE_CART

#VARVOL=5

%BLOCK POSITIONS_FRAC
C 0.0 0.0 0.0 # C1 % NUM=1
C 0.0 0.0 0.0 # C2 % NUM=1
%ENDBLOCK POSITIONS_FRAC

#MINSEP=1.3

KPOINTS_MP_SPACING 0.07
```

在本文件中，AIRSS指令`#VARVOL=5`指定所生成晶胞的体积为5立方埃，上下浮动5%。单位晶胞中存在两个碳原子，标记为`C1`和`C2`。

AIRSS指令`#MINSEP=1.3`设置原子间最小间隔为1.3埃。这对于轻元素来说不是必需的，但对于过渡金属来说，避免核心重叠以防止电子结构收敛不良很重要。

最后，CASTEP指令`KPOINTS_MP_SPACING`设置k点采样密度为0.07，对于金属（或接近金属）系统，0.05更合适。

文件`C2.param`包含CASTEP使用的DFT/松弛参数。有关这些参数的信息，请参阅CASTEP文档，或使用`castep --help <keyword>`。


**`C2.param`:**

```

```

## 运行示例

```console
$ airss.pl -press 100 -max 10 -seed C2
```

命令行选项`-press 100`将压力设置为100 GPa，`-max 10`告诉AIRSS生成10个结构，`-seed C2`给出seedfile的名称（`C2.cell`）。

由于未设置其他选项，AIRSS将默认使用CASTEP代码。

在运行过程中，可以使用`xmgrace`来绘制`.conv`文件以监测进度：

```console
$ xmgrace *.conv
```

这将产生每个结构在松弛过程中的焓与迭代次数的曲线图。焓以电子伏特(eV)为单位。

## 检查输出

运行后，我们可以使用cryan工具(`ca`)获得按能量排序的结构列表。注意，由于AIRSS的随机性，您的确切输出将与此处显示的输出不同：

```console

C2-90568-5971-5      100.00     4.699     0.000  2 C            Fd-3m      1
C2-90568-5971-7      100.01     4.698     0.002  2 C            Fd-3m      1
C2-90568-5971-8       99.99     4.698     0.002  2 C            Fd-3m      1
C2-90568-5971-2       99.99     4.697     0.003  2 C            Fd-3m      1
C2-90568-5971-1       99.94     4.698     0.003  2 C            Fd-3m      1
C2-90568-5971-6      100.00     4.695     0.005  2 C            Fd-3m      1
```

在这个很小的晶胞中，发现的大多数结构都是金刚石结构(空间群`Fd-3m`)。如果在较低压力(例如1 GPa)下重复搜索，将发现更多的石墨状结构。
