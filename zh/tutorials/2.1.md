在本例中，我们将探索100 GPa（或1 MBar）下碳的能量景观。作为对比，地球中心的压力约为350 GPa。

为了进行计算，我们生成碳结构，并使用GULP代码和Tersoff势来放松结构。

## 输入文件

首先，检查种子文件`C2.cell`：

```console
$ cat C2.cell

%BLOCK LATTICE_CART
1.709975 0 0
0 1.709975 0
0 0 1.709975
%ENDBLOCK LATTICE_CART

#VARVOL=5

%BLOCK POSITIONS_FRAC
C 0.0 0.0 0.0 # C1 % NUM=1
C 0.0 0.0 0.0 # C2 % NUM=1
%ENDBLOCK POSITIONS_FRAC

#MINSEP=1.3

KPOINTS_MP_SPACING 0.1
```

在此文件中，AIRSS指令`#VARVOL=5`指定生成的晶胞体积为5立方埃加减5%。晶胞中有两个碳原子，标记为`C1`和`C2`。AIRSS指令`#MINSEP=1.3`设置最小原子-原子分离为1.3埃。

文件`C2.lib`给出GULP所需的Tersoff势参数。


**`C2.lib`：**

```
#  C  : J. Tersoff, Phys. Rev. Lett., 61, 2879 (1988)
species
C  core 0.0
botwobody
C  core C  core 1393.600 346.7400 3.48790 2.2119 1.8000 2.1000
borepulsive
C  core C  core 0.0 3 0.72751 2.2119
boattractive theta
C  core C  core 0.00000015724 3 0.72751 2.2119  38049.0  4.3484 -0.57058
```

## 运行示例

```console
$ airss.pl -gulp -press 100 -max 10 -seed C2
```

命令行选项`-gulp`告诉AIRSS使用GULP代码，`-press 100`将压力设置为100 GPa，`-max 10`告诉AIRSS生成10个结构，`-seed C2`给出`.cell`输入文件的种子名称（`C2.cell`）。
