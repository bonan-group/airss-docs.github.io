在这个例子中，我们将使用随机搜索来寻找具有38个原子的Lennard-Jones团簇的基态，使用对称性。

```console
$ cat Al.cell

%BLOCK LATTICE_CART
20 0 0
0 20 0
0 0 20
#FIX
%ENDBLOCK LATTICE_CART

%BLOCK POSITIONS_FRAC
Al 0.0 0.0 0.0 # Al1 % NUM=38
%ENDBLOCK POSITIONS_FRAC

FIX_ALL_CELL : true

#MINSEP=1.5
#CLUSTER
#POSAMP=3.75
#SYMMOPS=2-5
#NFORM=1
```

随机生成的团簇将包含38个原子，并具有具有2到5个对称操作之间的随机选择的点群的对称性（`#SYMMOPS=2-5`）。如果省略 `NFORM=1`，默认行为是团簇中的原子数将为nops x 38，这不太可能在半径为3.75和最小间距为1.5的球体中满足。

**`Al.pp`：**

```
1 12 6 10.0
Al
# Epsilon
1
# Sigma
2
# Beta
1
```

```console
$ airss.pl -pp3 -cluster -max 200 -seed Al &
```
